Classification of wireless device location based on WiFi metadata

Guidance J.F. Chamberland, G. H. Huff
Description
The Wi-Fi metadata of mobile devices can be obtained by deploying a network of monitoring devices over Wi-Fi. Monitoring devices consist of network interface cards in monitoring mode and are capable of listening to all Wi-Fi packets in the network traffic. This study examines the problem of estimating the occupancy of a region based on Wi-Fi metadata and compares the performance obtained with the data gathered by monitoring system with directional antennas to that of the same system with isotropic antennas.

In this work, occupancy estimation is formulated as a classification problem, and logistic regression and radial basis function networks are employed as the classification schemes. The metadata from Wi-Fi packets combined with received signal strengths from the monitoring antennas act as input to the classifiers. This work considers a specific scenario of occupancy estimation. The classification performance of the system is assessed with the synthetic data at different noise levels in the RSSI values, and with the experimental data obtained from a testbed implementation. The field data are acquired in a line-of-sight environment.

This work introduces machine learning approaches for occupancy estimation based on Wi-Fi metadata and provides insight into the efficiency of directional antennas over isotropic antennas in the context of occupancy estimation. It also suggests that utilizing custom antenna designs based on the inference, shape and size of the target region can significantly improve the performance of a Wi-Fi monitoring system.
Keywords
Machine Learning, Classification, Logistic regression, Gaussian kernels, Support Vector Machines (SVM).

Evolutionary algorithms and their applications in Seating arrangement optimization

Contributors Pranay kumar,
Description
In this work, the optimization of seating arrangement of an examination hall is done by maximing the distance between the students that belong to the same subject. Given a room configuration (No. of rows and coloumns) and the number of students, the objective is to use biologically inspired optimization algorithms to propose the seating plan. Initially, the Particle Swarm Optimization is used to solve the problem at hand. But later different Soft & Evolutionary algorithms such as Ant Colony Optimization, Differential Evolution and Genetic Algorithm are applied.

Prediction Model for New Product Development Rate (NPDR)

Guidance Prof.G.Panda, Deputy Director, IIT Bhubaneswar
Description
This work aims at modelling a relationship between the New Product Development Rate (NPDR) of an industry and the factors affecting it, so that the companies can make intelligent investment in this area. Three industries - Electronics, Auto mobiles & Construction are considered for this work and prediction models are developed for each industry using the corresponding data.

The prediction of New Product Development Rate is analysed using Functional Link Artificial Neural Network (FLANN) with trigonometric Expansion. The FLANN model is trained with LMS algorithm and RLS algorithm to learn the weights associated with the model. The mean absolute percentage error (MAPE) with respect to actual New Product Development Rate is used as Performance index of the model. The general experimental procedure for three industries was discussed. Test results & simulations showed FLANN model with RLS training provides the best performance for Construction and Auto components industry and FLANN model with a neuron(sigmoid function) at the output provides the best performance for Electronics industry. This model has achieved Mean Average Percentage Error(MAPE) of less than 10% for each of these industries.
Keywords
Logistic regression, Radial Basis Functions (RBF), Multilayer perceptrons(MLP), Functional link artificial neural network (FLANN), LMS , RLS

Predictive model for diagnosis of Chronic Periodontal disease

Guidance Prof.G.Panda, Deputy Director, IIT Bhubaneswar
Description
This project aims to improvise the dental care in developing countries (where the dentist to population ratio stands much lower at 1:22000(like India) compared to the WHO guidelines of at least 1:7500) by developing methods that can be used in the absence of an expert clinician to accurately predict the disease condition given the patient symptoms. This project was done under the guidance of Prof. Ganapati Panda, Deputy Director, IIT Bhubaneswar.

The Clinical data of patients was obtained by experts in periodontics after examination of patients in a non-numerical format. The data was collected from patients reporting to the department of Periodontics and Oral Implantology, Institute of Dental Sciences, Siksha ‘O’ Anusandhan university, Bhubaneswar, India.

The Functional link artificial neural network (FLANN) is used to model the non-linear input-output relationship. The FLANN model substantially increases the non-linearity of the input elements by using functional expansion and it has faster convergence rate and lesser computational load. This model has achieved 99.11% accuracy with high sensitivity and specificity which makes it an excellent model in the field of diagnostic medicine.

For further details i.e., design of input data and features, development of the model: experimental conditions, training, validation and testing, performance results refer to the attached document.
Keywords
Functional link artificial neural network(FLANN), multilayered artificial neural networks (MLANN), LMS , RLS
